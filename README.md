## Build Setup Local

<p>Frontend></p>

```bash
# install dependencies
cd frontend
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```

<p>Backend></p>

```bash
# install dependencies
cd backend
$ npm install

# serve backend at localhost:5000
$ npm run start
```
